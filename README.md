# Teslo API
1.Clonar proyecto

2.Instalar los modulos de node
```
yarn install
````
3.Clonar el archivo **.env.template** y renombralo a **.env**

4.Renombrar las variables de entorno

5.levantar la base de datos
```
docker-compose up -d
```
6.Ejecutar Seed
```
localhost:3000/api/seed
```

7.Levantar el modo desarrollo
```
yarn start:dev
```
